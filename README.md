NotOpenWinStore
===============

Windows ストアアプリのアプリページから自動的にストアが起動しないようにします。  
Prevent opening the Windows Store automatically.

----

この拡張自体の説明は特にないので宣伝を。  
Windows ストアアプリ向け Twitter クライアント「 Mevy 」を制作しています。  
無料なのでもし良かったら。（広告が表示されます）
